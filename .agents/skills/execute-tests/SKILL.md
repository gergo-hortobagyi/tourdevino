# Execute Tests

## Overview

execute-tests

## When to Use

When the user asks to execute tests

## Instructions

1. Delete the agent_workspace/test-output.txt file
2. Read the app/TEST_GUIDE_APP.md or the backend/TEST_GUIDE_BACKEND.md depending on what tests are needed
3. Execute the tests, write their output to the agent_workspace/test-output.txt
4. DO NOT FIX ISSUES, this will be done by another agent
